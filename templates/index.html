Home Page

 

This project titled “COVID 19 Pandemic - a DBMS approach” aims to implement database concepts involving storing and analyzing the details concerning various aspects of the ongoing COVID-19 pandemic, and how such a database model may be useful for managing the pandemic.

In this project, we have developed a model that integrates a database system with web applications. The model aims to provide better tools for the management of population (healthy individuals) and the victims alike in a given area.
